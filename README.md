## 📁 Struktur Folder

```
koichi-project/
│
├── index.html          → Halaman utama website
├── /css
│    └── style.css      → Semua styling (CSS Variables, dark mode, responsive)
│
├── /js
│    └── script.js      → Interactivity (loader, navbar, animasi, dark mode)
│
├── /assets
│    ├── images/        → Tempatkan gambar di sini (foto team, dll.)
│    └── icons/         → Tempatkan ikon kustom di sini
│

```

## ✨ Fitur Website

| Fitur | Keterangan |
|---|---|
| 🌸 Japanese Minimal Design | Estetika Jepang modern — bersih, elegan, premium |
| 🌙 Dark Mode | Toggle antara light/dark, disimpan di localStorage |
| ⚡ Loader Animation | Japanese kanji loader animasi saat pertama buka |
| 📱 Fully Responsive | Mobile-first, tablet & desktop |
| 🔄 Scroll Animations | Fade-in smooth menggunakan IntersectionObserver |
| 🗂 Active Nav Link | Navbar otomatis highlight section yang aktif |
| 🧲 Magnetic Buttons | Efek magnetik halus pada hover tombol |
| ♿ Accessibility | Keyboard navigation, ARIA labels, reduced-motion |
| 🔍 SEO Friendly | Meta tags, semantic HTML5, proper heading hierarchy |

## 🎨 Design System

### Warna
| Token | Light | Dark |
|---|---|---|
| Background | `#FAFAF8` | `#0D0D0D` |
| Surface | `#FFFFFF` | `#1C1C1C` |
| Text | `#0D0D0D` | `#F0EEE8` |
| Accent Red | `#E60023` | `#E60023` |
| Muted Text | `#6B6860` | `#8A8880` |

### Typography
- **Heading:** Playfair Display / Noto Serif JP (elegant serif)
- **Body:** Poppins (modern sans-serif)
